HTML не үшін қажет?

HTML (HyperText Markup Language) — бұл веб-беттердің құрылымын жасауға арналған негізгі тіл. Ол мәтін, сурет, сілтеме, кесте және басқа элементтерді орналастыруға мүмкіндік береді. Әрбір веб-сайттың негізі HTML болып табылады.

HTML код үлгісі

<!DOCTYPE html>
<html lang="kk">
  <head>
    <meta charset="UTF-8">
    <title>Мысал бет</title>
  </head>
  <body>
    <h1>Сәлем әлем!</h1>
    <p>Бұл HTML мысалы.</p>
  </body>
</html>

CSS дегеніміз не?

CSS (Cascading Style Sheets) — веб-беттердің сыртқы көрінісін безендіруге арналған тіл.

CSS код үлгісі

body {
    background-color: #f4f4f4;
    color: #333;
}
h1 {
    color: #35424a;
}

Негізгі CSS қасиеттері

color — мәтін түсі

Элементтің мәтін түсін анықтайды. Мысал: color: red;

background-color — фон түсі

Элементтің фон түсін орнатады. Мысал: background-color: #f4f4f4;

font-size — қаріп өлшемі

Мәтін өлшемін орнатады. Мысал: font-size: 16px;

margin — сыртқы аралық

Элементтің шекарасынан тыс кеңістік жасайды. Мысал: margin: 10px;

padding — ішкі аралық

Элементтің ішінде, шекара мен мазмұн арасында кеңістік жасайды. Мысал: padding: 10px;

border — шекара

Элементтің айналасына жиек қосады. Мысал: border: 1px solid #333;

display — көрсету түрі

Элементтің бетте қалай көрсетілетінін анықтайды. Мысал: display: flex;

flex — flex-контейнер параметрлері

Икемді орналасу үшін қолданылады. Мысал: display: flex; flex-direction: row;

text-align — мәтінді туралау

Мәтінді көлденеңінен туралайды. Мысал: text-align: center;

Негізгі HTML тегтері

<h1>–<h6> — тақырыптар

Әр түрлі деңгейдегі тақырыптарды жасау үшін қолданылады.
Мысал: <h1>Тақырып</h1>

<p> — абзац

Мәтін абзацын жасайды.
Мысал: <p>Абзац мәтіні</p>

<a> — сілтеме

Гиперсілтеме жасайды.
Мысал: <a href="https://example.com">Сілтеме</a>

<img> — сурет

Сурет енгізеді.
Мысал: <img src="image.jpg" alt="Сипаттама">

<ul>, <ol>, <li> — тізімдер

<ul> — белгіленген тізім, <ol> — нөмірленген, <li> — тізім элементі.
Мысал: 

<ul>
  <li>Тармақ 1</li>
  <li>Тармақ 2</li>
</ul>

<div> — контейнер

Блоктық элемент — басқа элементтер үшін контейнер.
Мысал: <div>Контент</div>

<span> — жолдық контейнер

Мәтіннің бөлігін ерекшелеу үшін жолдық элемент.
Мысал: <span>Мәтін</span>

<table> — кесте

Кесте жасайды.
Мысал: 

<table>
  <tr><td>Ұяшық</td></tr>
</table>

<form> — форма

Деректер енгізу үшін форма жасайды.
Мысал: 

<form>
  <input type="text">
  <button>Жіберу</button>
</form>
